XVM Updater

World of Tanks' eXtended Visualization Mod one click installer/updater

Download

XVM Updater is a little tool enabling you to easily install (and update to its latest version) the eXtended Visualization Mod (XVM).

This app will automatically detect your World of Tanks installation and download the required files.

It also contains a few useful features like saving your current configuration or downloading a customized one.

How to install XVM?

  • Download XVM Updater
  • Open the XVM_Updater.exe file
  • (optional) Select the configuration you want or choose to keep your existing one
  • Click Install / Update
  • (optional) Enable statistics on the XVM website (instructions below)
